# Бюро инженерно-строительных консультаций

Сайт компании, предоставляющей услуги технадзора и проектирования.

## 🛠️ Технологии

- **Astro** — фреймворк для статических сайтов
- **Tailwind CSS 4** — стилизация
- **Keystatic** — CMS для управления контентом
- **Resend** — отправка Email
- **Netlify** — хостинг и CI/CD

## 📂 Структура

```
├── content/           # Контент (Keystatic: услуги, проекты, настройки)
├── public/            # Статические файлы (изображения, robots.txt)
└── src/
    ├── components/    # Astro компоненты
    ├── layouts/       # Шаблоны страниц
    ├── pages/         # Страницы и API роуты
    │   └── api/       # API функции (send.ts)
    └── styles/        # Глобальные стили
```

## 🚀 Запуск

```bash
# Установка зависимостей
npm install

# Режим разработки
npm run dev

# Сборка для продакшена
npm run build
```

## ✏️ Редактирование контента

Админка доступна по адресу: `/keystatic`
(В режиме разработки запускается локально)

Разделы:
- **Услуги** — управление списком услуг
- **Проекты** — портфолио
- **Настройки** — контакты, соцсети
- **Hero** — главный экран
- **О компании** — блоки "О нас"

## 🌐 Деплой

Проект настроен для деплоя на **Netlify**.

1. Подключите репозиторий в Netlify.
2. Build command: `npm run build`
3. Publish directory: `dist`
4. Не забудьте указать Environment Variables в настройках сайта.
