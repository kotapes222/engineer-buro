# Бюро инженерно-строительных консультаций

Сайт компании, предоставляющей услуги технадзора и проектирования.

## 🛠️ Технологии

- **Astro** — фреймворк для статических сайтов
- **Tailwind CSS 4** — стилизация
- **Keystatic** — CMS для управления контентом

## � Структура

```
├── content/           # Контент (Keystatic)
│   ├── services/      # Услуги
│   ├── projects/      # Проекты
│   └── *.json         # Настройки, Hero, О компании
├── public/
│   ├── api/           # PHP скрипты (отправка форм)
│   └── images/        # Изображения
└── src/
    ├── components/    # Компоненты
    ├── layouts/       # Шаблоны
    ├── pages/         # Страницы
    └── styles/        # CSS
```

## 🚀 Запуск

```bash
# Установка зависимостей
npm install

# Режим разработки
npm run dev

# Сборка для продакшена
npm run build
```

## ✏️ Редактирование контента

Админка: `/keystatic`

- **Услуги** — добавление/редактирование услуг
- **Проекты** — портфолио работ
- **Настройки** — контакты компании
- **Hero** — главный баннер
- **О компании** — текст и преимущества

## 📧 Отправка форм

Файл: `public/api/send.php`

Настройте:
```php
$to_email = "your@email.ru";
$telegram_token = "BOT_TOKEN";
$telegram_chat_id = "CHAT_ID";
```

## 🌐 Деплой

Хостинг: Beget (PHP)

1. `npm run build`
2. Загрузить содержимое `dist/` + папку `public/api/`
